# Personality Receipt — Fun Personality Test as a Store Receipt

Fun personality test that prints your traits as a store receipt. Take the quiz and get a shareable personality receipt in seconds.

**Live site: [https://receipt.csskey.com](https://receipt.csskey.com)**

## Features

- **Personality quiz** — Quick questions about your traits
- **Receipt format** — Traits printed as a store receipt
- **Trait breakdown** — Strengths, weaknesses, and quirks
- **Shareable receipt** — Download or share your result
- **Fun categories** — MBTI-inspired, enneagram, and more

## Tech Stack

- **Frontend:** React, TypeScript, Vite
- **Styling:** Tailwind CSS
- **SEO:** react-helmet-async, JSON-LD Schema
- **Hosting:** Cloudflare Workers

## Quick Start

```bash
# Install dependencies
bun install

# Start dev server
bun run dev

# Build for production
bun run build
```
